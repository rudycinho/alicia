Funciones de Escala y Crecimiento Exponencial

Alicia y los Cambios de Tamaño: Modelando la Escala con Octave

Enunciado del problema. Alicia experimenta cambios de tamaño tras ingerir sustancias mágicas. Su altura $h(t)$ se modela como:

$$ h(t) = h_0\,e^{kt}, $$ donde $h_0$ es la altura inicial en $t=0$ y $k$ la tasa de crecimiento (positiva o negativa). Se pide:
  1. Hallar $k$ si a los $T$ minutos la altura es $h(T)$.
  2. Graficar $h(t)$ en $t\in[0,T]$.

Resolución.

$$ h(T) = h_0 e^{kT} \Rightarrow k = \frac{1}{T}\ln\left(\frac{h(T)}{h_0}\right). $$ Así, $$ h(t) = h_0\exp\left(\tfrac{t}{T}\ln\tfrac{h(T)}{h_0}\right). $$

El siguiente bloque contiene el código en Octave para calcular y graficar $h(t)$:

// Parámetros
h0 = 1.2;      % Altura inicial (m)
hT = 2.4;      % Altura a tiempo T (m)
T  = 5;        % Tiempo final (minutos)

// Cálculo de k
k = (1/T) * log(hT / h0);

// Vector de tiempo
t = linspace(0, T, 100);

// Modelo exponencial
h = h0 * exp(k * t);

// Graficar
figure;
plot(t, h, 'LineWidth', 2);
xlabel('Tiempo (minutos)');
ylabel('Altura (m)');
title('Crecimiento Exponencial de Alicia');
grid on;

// Guardar figura
print('alicia_crecimiento.png', '-dpng');
